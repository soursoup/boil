[![Code Climate](https://api.codeclimate.com/v1/badges/b56e0c5a0856da0c35ba/maintainability)][codeclimate]

## Installation

Use pip or easy_install:

```bash
$ pip install boil
```

## Usage

Initialize new project:

```bash
$ boil new [PLATE]
```

Examples:

```bash
# New Python package
$ boil new python-package

# New Django app
$ boil new django-app

# New Rails 5 app
$ boil new rails-app

# New Ruby gem
$ boil new gem

# New Ansible role
$ boil new ansible-role

# New plate
$ boil new plate
```
